# Numpy-and-Pandas
# 📊 Sales Data Analysis using NumPy & Pandas

A beginner-friendly data analytics project demonstrating how NumPy and Pandas can be used to analyze sales data and generate meaningful business insights.

---

## 📌 Features

- Data loading
- Data cleaning
- Statistical analysis
- Revenue calculation
- GroupBy operations
- Data visualization

---

## 🛠 Technologies

- Python
- NumPy
- Pandas
- Matplotlib
